Reduced Basic cross-compiler

v0.02 for the Элeктpoникa БK0011

It supports a lot of keywords from BK0010 ROM Basic and even several keywords from BK0011 ROM Basic. It also provides the several new features: byte file io, full support for the string type, etc. However some old features are missed: the real numbers, graphics, ...

The compiler is available via sources (23K),
a small exe-file for Microsoft Windows without support of the optimization (249K),
the complete version for the Microsoft Windows (1748K).
