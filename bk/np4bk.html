This is a lightweight and simple fullscreen text editor written in Basic for ANDOS. It supports the work with any plain text files. It may edit, in particular, a text with any line length. Just run NP4.COM

The distribution disk contains sources, documentation and a compiled executable. This executable is produced with RBK-Basic cross-compiler
