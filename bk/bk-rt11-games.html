Hесколько подсказок для работы с RT11

Команда DIR покажет список всех файлов на диске. Запуск игры производится командой RUN ИМЯ-ИГРЫ, конечный .SAV указывать не нужно. Например, RUN STALK. Список игр идёт далее. Команда CLS чистит экран. В некоторых играх следует включать русский алфавит клавишей РУС. Плохо, что на эмуляторах БК клавиши цифровой клавиатуры не производят цифры, а многие игры используют именно цифровую клавиатуру.

На диске #2 есть игры на бейсике, для их запуска нужно сначала загрузить бейсик командой BASIC. Затем нужно разобраться с подключением опциональных функций. Для большинства случаев набираем ALL, для некоторых - NONE. Затем грузим игру командой OLD ИМЯ-ИГРЫ, конечный .BAS указывать не нужно. Запускаем командой RUN. Выход из бейсика в ОС производится командой BYE.

шахматы, используется древняя описательная нотация, 
когда первая вертикаль называется QR, а не A. Это один из самых ранних
вариантов шахмат, середины 70-х. Команды:
  BD - board - покажи доску
  D= - set depth - установи глубину анализа (1-5, по умолчанию 4), чем больше глубина, тем дольше компьютер думает
  EX - exit - выход
  P= - установка номера хода, затем в следующих 8 строках задаём позицию на доске в упрощенной нотации Форсайта-Эдвардса (FEN), например, задача на мат в три хода

      1WR5WP  - *R*****P - белые ладья и пешка
      8       - ******** - пустая горизонталь
      8       - ********
      5WP2    - *****P** - белая пешка
      3WB1WQ2 - ***B*Q** - белые слон и ферзь
      8       - ********
      2WK5    - **K***** - белый король

  PN - play neither - компьютер отключается (так на старте)
  PS - play show - информация о параметрах
  PW - play white - компьютер играет за белых. Можно одновременно с PB, тогда компьютер играет за обоих игроков
  RE - reset - рестарт
  SK - skip - пропусти ход
  T= - установка параметра глубины печати на принтере LP: дерева игры, любые значения кроме 0 приводили на моих тестах к аварии
  W= - width - задаёт через запятую первые параметры (их число равно D-1) для дерева анализа, например,  W=10,10,10 задает три первых ширины просмотра на каждом уровне дерева просмотра в 10 (по умолчанию 8). Ширина не может быть большей 99
  P/K2-K4 или P-K4 - ход белой пешки E2-E4 в стандартной нотации
  P/K2-K4 или P-K4 - ход черной пешки E7-E5 в стандартной нотации
  P/K4*P/Q5 - взятие пешки на D5 с E4 в стандартной нотации, описать взятие проходной не получилось
  OO - короткая рокировка
  OOO - длинная рокировка
  выход по СУ/С
