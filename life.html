JOHN CONWAY'S GAME OF LIFE, FOR COMMODORE 264 SERIES


This game was created in 1989. Initial info screen was added in 1991. The conversion to the PRG format was done in 2003. The border status sets transparency of border — 0 means impenetrable border, i.e. status 0 sets rectangular field and status 1 sets ringlike area. The field size is 80x100 cells (cellular-automatons). Create new life!
