Superfast Mandelbrot generator v3

It is based on an algorithm published on pouet in 2021. Its implementation uses 16 colors. Generated pictures are 128x256 raster images. This algoritm implementation is among fastest known Mandelbrot set computation routines for the ARM.

An archive-file with executables is here. As an extra it contains a fullscreen (896x352, 800x600, 1056x256, ..., 16/256 colors) Mandelbrot generator v1 which can draw, for instance the next image (640x512, 256 colors).

Sources are available on github.
